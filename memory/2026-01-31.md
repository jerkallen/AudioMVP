# 2026-01-31 - LearnWithDaniel 项目启动

## 项目背景

为 Daniel（初二学生）开发英语单词学习助手，帮助他更高效地背单词。

## 核心需求

1. **拍照输入单词表** - 使用大模型 OCR 识别
2. **两阶段记忆法** - 选释义 → 拼写
3. **智能复习** - 错词重复 + 遗忘曲线
4. **学习进度追踪** - 统计和可视化

## 技术方案

**后端：** Flask 3.0
**数据库：** SQLite
**AI能力：** 通义千问视觉模型（qwen-vl-plus）
**前端：** 响应式设计（电脑/移动端自适应）
